## TRACE APP - Setup Instructions

1. Clone: git clone https://github.com/pendyalapriya11-gif/TRACE_APP.git
2. cd TRACE_APP
3. npm install
4. Create a .env file in root with:
   DB_HOST=localhost
   DB_USER=root
   DB_PASSWORD=yourpassword
   DB_NAME=trace_db
   JWT_SECRET=yoursecret
   NODE_ENV=development
5. npm run dev
6. Open frontend/index.html in browser
